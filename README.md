AgroAssist
AgroAssist is a comprehensive web-based platform designed for farmers to make informed agricultural decisions. It includes features like crop recommendation, weather forecasts, market analysis, and the latest agriculture-related news.


Features
Login/Signup System: Allows users to register and authenticate themselves.
Crop Recommendation: Suggests crops based on soil and weather conditions.
Market Analysis: Provides historical, current, and future crop price data.
Weather Forecast: Displays real-time and forecasted weather for agricultural planning.
News: Aggregates the latest news articles related to agriculture.
Technologies
Frontend: React, React Router DOM, Axios
Backend: Node.js, Express.js, MongoDB (Mongoose)
API Integration: Weather API, News API
